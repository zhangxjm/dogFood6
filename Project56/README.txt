Non-Heritage Cultural E-commerce Private Domain Operations System
============================================================

Quick Start:
1. Run init.bat (first time only)
2. Run start.bat

Admin Account: admin / admin123

Demo Users: zhangsan / test123456
            lisi / test123456
            wangwu / test123456

Services:
- Frontend: http://localhost:3000
- Backend API: http://localhost:8000/api/
- Django Admin: http://localhost:8000/admin

Tech Stack:
- Frontend: SolidJS + Solid Router + TailwindCSS + Vite
- Backend: Django 5 + Django REST Framework
- Database: SQLite
- Cache: Redis (Docker)

Features:
- Product Customization
- Member Management (Levels, Points, Tasks)
- Private Domain Campaigns (Coupons, Group Buy, Share)
- Personalized Recommendations (Content-based + Collaborative Filtering)
