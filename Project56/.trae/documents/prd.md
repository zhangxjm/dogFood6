## 1. 产品概述
非遗文创电商私域运营系统，旨在为非物质文化遗产文创产品提供从展示、定制、交易到私域会员运营的一站式平台。通过个性化推荐引擎与私域流量转化机制，帮助商家深度经营用户关系，实现文创品的高效传播与销售转化。
- 解决非遗文创品销售渠道单一、用户粘性低、私域运营手段缺乏等痛点
- 目标用户为非遗文创品消费者、私域会员及运营管理员

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 手机号注册 | 浏览商品、加入会员、下单定制、参与活动 |
| 管理员 | 后台指定 | 商品管理、会员管理、订单管理、数据看板 |

### 2.2 功能模块
1. **首页**: 品牌展示、推荐商品、私域活动入口、新品首发
2. **文创品商城**: 分类浏览、搜索、商品详情、个性化推荐
3. **文创品定制**: 在线定制流程、模板选择、个性化参数配置、预览确认
4. **会员中心**: 会员等级、积分体系、专属权益、成长任务
5. **私域运营**: 社群管理、优惠券发放、拼团活动、裂变分享
6. **订单管理**: 购物车、下单、支付状态、订单跟踪
7. **管理后台**: 数据看板、商品CRUD、会员分析、订单处理

## 3. 核心流程

### 3.1 文创品定制流程
用户浏览商品 → 选择定制 → 选择模板 → 配置个性化参数 → 实时预览 → 加入购物车/直接下单 → 支付 → 等待制作 → 收货

### 3.2 私域转化流程
用户访问 → 注册成为会员 → 参与签到/任务积累积分 → 领取优惠券 → 分享裂变邀请新用户 → 新用户注册转化 → 会员等级提升

### 3.3 个性化推荐流程
用户行为采集（浏览/收藏/购买）→ 特征提取 → 协同过滤推荐计算 → 首页/详情页推荐位展示 → 用户点击反馈 → 推荐模型迭代

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：朱砂红（#C41A1A）+ 黛青色（#1A3A5C），辅以金色（#D4A843）点缀
- 按钮风格：圆角微弧（8px），主按钮朱砂红填充，次按钮黛青描边
- 字体：标题使用思源宋体风格，正文使用系统默认中文字体，字号16px正文 / 24px标题
- 布局风格：卡片式布局，顶部导航栏，侧边分类菜单
- 图标风格：线性图标搭配非遗文化装饰纹样

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 品牌横幅 | 全宽轮播，渐变叠层，中国风纹样装饰 |
| 首页 | 推荐商品区 | 3列卡片网格，悬浮阴影动效，hover缩放 |
| 首页 | 私域活动入口 | 横向滚动卡片，渐变背景，倒计时动画 |
| 文创品商城 | 分类导航 | 左侧固定分类栏，选中态金色底边 |
| 文创品商城 | 商品列表 | 响应式网格，懒加载图片，标签徽章 |
| 商品详情 | 商品展示 | 左图右详情，图片放大镜，非遗故事区 |
| 文创品定制 | 定制面板 | 左侧预览画布，右侧配置面板，实时渲染 |
| 会员中心 | 等级卡片 | 渐变背景会员卡，成长值环形进度条 |
| 私域运营 | 活动列表 | 卡片式活动展示，标签状态，参与人数 |
| 管理后台 | 数据看板 | 图表卡片组，折线图/饼图，数据指标卡 |

### 4.3 响应式设计
- 桌面优先设计，最小宽度1200px
- 平板端（768-1200px）：侧边栏折叠为汉堡菜单，2列商品网格
- 移动端（<768px）：单列布局，底部导航栏，适配小程序SDK

### 4.4 小程序SDK对接
- 页面结构兼容小程序WebView嵌入
- 关键交互使用小程序JS-SDK能力（分享、支付）
- 响应式布局适配小程序容器尺寸
