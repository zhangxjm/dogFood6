## 1. 产品概述

成人自考学习资料整理平台，为自考学员提供一站式的备考资料管理、学习打卡与进度追踪服务，帮助学员高效备考、科学规划学习进度。

- 解决自考学员资料分散、缺乏系统管理的问题
- 目标用户：参加成人高等教育自学考试的在职学员

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 学员 | 用户名注册 | 浏览资料、上传资料、打卡、追踪进度 |

### 2.2 功能模块

1. **首页仪表盘**：学习统计概览、今日打卡状态、最近上传资料、学习进度总览
2. **资料中心页**：分类浏览备考资料、搜索筛选、上传分享资料
3. **打卡记录页**：每日打卡、打卡日历、连续打卡统计
4. **进度追踪页**：科目学习进度、时间线视图、学习时长统计

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页仪表盘 | 学习统计卡片 | 显示总资料数、已学科目数、连续打卡天数、学习总时长 |
| 首页仪表盘 | 今日打卡状态 | 显示今日是否已打卡，快速打卡入口 |
| 首页仪表盘 | 最近上传资料 | 展示最近5条上传的资料列表 |
| 首页仪表盘 | 学习进度总览 | 各科目进度环形图 |
| 资料中心页 | 分类导航栏 | 按考试科目、资料类型（教材、真题、笔记、视频）分类 |
| 资料中心页 | 资料列表 | 卡片式展示资料，支持搜索筛选 |
| 资料中心页 | 资料上传弹窗 | 上传资料文件，填写名称、分类、描述 |
| 打卡记录页 | 打卡按钮 | 一键打卡，记录打卡时间和备注 |
| 打卡记录页 | 打卡日历 | 日历视图显示打卡记录，标记已打卡日期 |
| 打卡记录页 | 打卡统计 | 连续打卡天数、累计打卡天数、本月打卡率 |
| 进度追踪页 | 科目进度列表 | 各科目学习进度条、完成百分比 |
| 进度追踪页 | 更新进度 | 更新科目学习进度、添加学习记录 |
| 进度追踪页 | 学习时长统计 | 柱状图展示每日/每周学习时长 |

## 3. 核心流程

用户打开平台 → 首页查看学习概况 → 进入资料中心浏览/上传资料 → 每日打卡记录学习 → 更新科目学习进度 → 查看进度统计

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深青蓝（#0F766E）+ 琥珀黄（#F59E0B）作为强调色
- 按钮风格：圆角按钮，带微妙阴影，hover 时轻微上浮
- 字体：思源黑体（Noto Sans SC）为主体字体，数字使用等宽风格
- 布局风格：左侧导航栏 + 右侧内容区，卡片式内容布局
- 图标风格：线性图标，来自 lucide-react

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页仪表盘 | 学习统计卡片 | 4列网格卡片，渐变背景，大号数字，小号标签 |
| 首页仪表盘 | 今日打卡 | 居中大按钮，打卡后变为已完成状态带动画 |
| 首页仪表盘 | 最近资料 | 水平卡片列表，缩略图+标题+标签 |
| 首页仪表盘 | 进度总览 | 环形进度图，各科目色块区分 |
| 资料中心页 | 分类导航 | 水平标签栏，选中态带下划线高亮 |
| 资料中心页 | 资料列表 | 3列网格卡片，封面图+标题+分类标签+上传时间 |
| 资料中心页 | 上传弹窗 | 居中模态框，拖拽上传区域+表单字段 |
| 打卡记录页 | 打卡日历 | 月历网格，已打卡日期带绿色圆点标记 |
| 打卡记录页 | 统计面板 | 3个统计数字卡片+打卡率进度条 |
| 进度追踪页 | 科目进度 | 垂直列表，每行科目名+进度条+百分比 |
| 进度追踪页 | 时长统计 | 柱状图，7天或30天切换 |

### 4.3 响应式设计

- 桌面优先设计，适配1280px及以上屏幕
- 平板适配：768px-1279px，导航栏折叠为汉堡菜单
- 移动端适配：768px以下，单列布局，底部导航栏
